📤 Passo a passo para enviar sua declaração ao sistema oficial

Agora que você gerou o documento, veja como enviar corretamente para a Receita Federal:

Passo 1: Baixe o programa oficial ou acesse o site

Você tem duas opções:

  • 🔹 Baixar o programa no seu computador: Clique aqui para acessar o site oficial
  • 🔹 Usar o site online, sem instalar nada: acesse o portal "e-CAC" ou o serviço "Meu Imposto de Renda"

Passo 2: Faça login

Você vai precisar de:
• CPF
• Senha do gov.br (se não tiver, é só criar — é rápido e gratuito)

Passo 3: Importar os dados

No programa ou site da Receita, procure a opção "Importar declaração" e escolha o arquivo que você gerou neste site. Ele vai preencher tudo automaticamente!

Passo 4: Conferir tudo

Veja rapidamente se todos os dados estão corretos — se tiver algo errado, é só alterar direto lá.

Passo 5: Enviar

Clique em "Entregar declaração" e pronto! Guarde o recibo que o sistema gerar — ele é a prova que você enviou.

📌 Dica importante

Sempre envie antes do prazo acabar! Se enviar depois, pode ter multa.
